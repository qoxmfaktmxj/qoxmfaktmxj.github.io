오라클 배치 확인

프로시저 돌리고 있어서 해당 부분 확인 하기 위해 메모



 

\-\-등록된 job

SELECT \* FROM USER\_SCHEDULER\_JOBS;

\-\-현재 running중인 job들의정보

SELECT \* FROM USER\_SCHEDULER\_RUNNING\_JOBS;

\-\-job의 log

SELECT \* FROM USER\_SCHEDULER\_JOB\_LOG ORDER BY LOG\_DATE DESC;;

\-\-job의수행된정보및Error 정보 

SELECT \* FROM USER\_SCHEDULER\_JOB\_RUN\_DETAILS ORDER BY LOG\_DATE DESC;

\-\- 등록된 Program

SELECT \* FROM USER\_SCHEDULER\_PROGRAMS; 

\-\-등록된 스케쥴러 

SELECT \* FROM USER\_SCHEDULER\_SCHEDULES;
